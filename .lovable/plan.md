
# 🎛️ خطة بناء لوحة تحكم الأدمن - تطبيق قطع غيار السيارات

## المرحلة 1: البنية الأساسية والتخطيط
- إنشاء Layout خاص بلوحة التحكم مع Sidebar قابل للطي
- إنشاء صفحة Dashboard رئيسية بإحصائيات سريعة (KPIs): إجمالي الطلبات، الإيرادات، العملاء، المنتجات
- رسوم بيانية (Recharts): مبيعات أسبوعية/شهرية، أكثر المنتجات مبيعاً
- Routing خاص بمسارات `/admin/*`

## المرحلة 2: إدارة المنتجات
- جدول المنتجات مع بحث وفلترة وترتيب
- نموذج إضافة/تعديل منتج (اسم، سعر، صورة، فئة، ماركة سيارة، توافق)
- إدارة الفئات والتصنيفات
- إدارة المخزون (كميات، تنبيهات نفاد)

## المرحلة 3: إدارة الطلبات والمدفوعات
- جدول الطلبات مع فلترة حسب الحالة (جديد، قيد المعالجة، مشحون، مكتمل، ملغي)
- تفاصيل الطلب مع تغيير الحالة
- التحقق من إيصالات التحويل البنكي (قبول/رفض)
- سجل المدفوعات وحالاتها

## المرحلة 4: إدارة العملاء
- جدول العملاء مع فلترة حسب النوع (عادي، ورشة، حكومي، جملة)
- تفعيل/إيقاف الحسابات
- مراجعة مستندات التحقق (المستندات الحكومية، السجلات التجارية)
- سجل مشتريات كل عميل

## المرحلة 5: إعدادات التوصيل والعملات
- إدارة مناطق التوصيل ورسومها
- تعديل أسعار صرف العملات (SAR, YER, OMR)
- إدارة الصرافات وحساباتها

## المرحلة 6: إدارة المرتجعات والإشعارات
- جدول طلبات الإرجاع مع مراجعة وقبول/رفض
- إرسال إشعارات للعملاء
- سجل الإشعارات المرسلة

## المرحلة 7: التقارير والإحصائيات
- تقارير المبيعات (يومي/أسبوعي/شهري)
- تقارير المنتجات الأكثر مبيعاً
- تقارير العملاء والمناطق
- تصدير البيانات (CSV)

---
**التصميم**: نفس الهوية البصرية (كحلي Navy)، Sidebar + Header، جداول وبطاقات متجاوبة، Dark mode
